Küsitluste vastamise ja tegemise lehekülg

Pealehel index.php on nähtaval aktiveeritud päevaküsimus, millele saab ühe IP pealt vastata vaid ühe korra.
Päevatulemuste lehel näeb käesoleva küsitluse tulemusi.
Küsitluste lehel saab teha uue küsimustiku kahe või kolme vastusevariandiga.
Küsitluste lehel saab aktiveerida päevaküsimusi ning korraga saab olla aktiivne ainult üks päevaküsimus.
Lisaks saab kustutada olemasolevaid küsitlusi ning vaadata nende tulemusi.

Rakenduse seadistamiseks tuleb config.php failis ära muuta host, username, password ja andmebaasi nimi.
Lisaks tuleb init.sql-i failis seadistada ära andmebaasi nimi, kuhu tabel luuakse.

Esmalt tuleb käivitada install.php ning seejärel saab minna index.php lehele.
